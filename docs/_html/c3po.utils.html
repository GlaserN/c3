c3po.utils.tf_utils.tf_evaluate(tf_obj_list, t, args)¶

This function is a 1:1 replication of qutips Qobj.evaluate() function but modified for the usecase with tensorflow objects. For more information see: http://qutip.org/docs/latest/modules/qutip/qobj.html#Qobj.evaluate

Evaluate a time-dependent tf.Tensor obj in list format. For example,

tf_obj_list = [H0, [H1, func_t]]

is evaluated to

tf.Tensor(t) = H0 + H1 * func_t(t, args)

and

tf_obj_list = [H0, [H1, ‘sin(w * t)’]]

is evaluated to

tf.Tensor(t) = H0 + H1 * sin(args[‘w’] * t)

Parameters

  • tf_obj_list (list) – A nested list of tf.Tensor obj instances and corresponding time-dependent coefficients.

  • t (float) – The time for which to evaluate the time-dependent Qobj instance.

  • args (dictionary) – A dictionary with parameter values required to evaluate the time-dependent tf.Tensor intance.

Returns

output – A tf.Tensor instance that represents the value of tf_obj_list at time t.

Return type

tf.Tensor
